Live TV Presidential Election Analysis
of the Republic of The Gambia

This project is special to me as it is about national service. I was contracted by the National TV Station of The Gambia (my country) to help broadcast on the national TV Live results of 2021 presidentail election.

Since it is Live TV, I needed to set up the refresh every 30 seconds or less; and because I don't have Premium Capacity license, I use Direct Query to connect my Azure SQL Server to Power BI and created a Power App which feeds the server with live results data. Finally, I use the Play Axis Custom Visual to set the refresh at every 30 seconds which amazingly refreshes the report in the Power BI Service every time there is live data without any restriction and displays it on the TV.
